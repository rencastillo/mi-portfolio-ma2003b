# Portfolio Overview

## Introducción
Este portfolio integra tres casos de estudio principales de análisis multivariante.

## Casos de Estudio

### 1. Factor Analysis
- Objetivo: Identificar variables latentes
- Métodos: Extracción de factores, rotación

### 2. Discriminant Analysis
- Objetivo: Clasificación predictiva
- Métodos: LDA, funciones discriminantes

### 3. Cluster Analysis
- Objetivo: Segmentación de datos
- Métodos: K-means, análisis jerárquico
