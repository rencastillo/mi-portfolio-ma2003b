# mi-portfolio-ma2003b

## Descripción General del Portfolio

Este repositorio contiene el portfolio de análisis multivariado para el curso MA2003B. El portfolio incluye tres casos de estudio que demuestran diferentes técnicas de análisis estadístico multivariado.

## Estructura del Proyecto

```
mi-portfolio-ma2003b/
├── README.md                          # Descripción general del portfolio
├── LICENSE                            # Licencia MIT
├── .gitignore                         # Configuración para Python
├── requirements.txt                   # Dependencias del proyecto
│
├── case-01-factor-analysis/           # Análisis Factorial
├── case-02-discriminant-analysis/     # Análisis Discriminante
├── case-03-cluster-analysis/          # Análisis de Clústeres
│
├── portfolio-summary/                 # Resumen integrador del portfolio
│   ├── PORTFOLIO_OVERVIEW.md
│   ├── LESSONS_LEARNED.md
│   └── METHODOLOGY_COMPARISON.md
│
└── presentation/                      # Presentación final
```

## Casos de Estudio

### Caso 1: Análisis Factorial
Aplicación de análisis factorial para identificar factores latentes en datos de satisfacción del cliente.

### Caso 2: Análisis Discriminante
Aplicación de análisis discriminante para clasificación y predicción.

### Caso 3: Análisis de Clústeres
Aplicación de técnicas de clustering para segmentación de datos.
