# Case 01: Factor Analysis

## Descripción
Este caso de estudio explora el análisis factorial aplicado a datos de satisfacción del cliente.

## Hallazgos Clave
- Estructura factorial identificada
- Variables latentes extraídas
- Varianza explicada
